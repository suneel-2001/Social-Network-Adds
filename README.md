📌 Project Overview
This project implements a Support Vector Machine (SVM) model to predict whether a user will purchase a product based on social network advertisement data.
The dataset contains user demographic information such as:
 - Age
 - Estimated Salary
 - Gender
 - Purchased (Target Variable)
The goal is to build a classification model that predicts whether a user will purchase a product.

🎯 Problem Statement
Given user demographic details, predict:
Will the user purchase the advertised product? (Yes/No)
This is a Supervised Machine Learning – Classification Problem.
